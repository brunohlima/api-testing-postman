# 🧪 Testes de API - ServeRest

Projeto de validação de API utilizando Postman.

## 🚀 O que foi testado

- Criação de produto (POST)
- Listagem de produtos (GET)
- Deleção de produto (DELETE)

## 🛠️ Ferramentas

- Postman
- ServeRest (API fake)

## 📁 Estrutura

- /postman → Collection exportada
- /prints → Evidências dos testes
